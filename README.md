# AI Programming Project - Rule-Based Chatbot

**Name:** Benedict Sau  
**Registration Number:** 250594DAI 
**Project Title:** AI Chatbot - Rule-Based  
**Short Description:**  
A simple Python chatbot that answers frequently asked questions about Artificial Intelligence (AI) and university-related queries. The chatbot uses rule-based logic (if-else and keyword matching) to respond to user inputs. It handles at least 5 different user intents, providing polite and informative responses.  

---

## **Features / User Intents**
1. **Greetings** – Responds to "Hi", "Hello", or "Hey".  
2. **Course Information** – Answers queries about available courses.  
3. **AI Basics** – Explains what AI (Artificial Intelligence) is.  
4. **University Information** – Provides the university location or details.  
5. **Farewell** – Responds politely when the user says "Bye" or "Goodbye".  
6. **Fallback** – Handles unknown inputs politely.
